# Betonwerken Careye

De volledige CMS-website wordt automatisch vanuit het projectpakket gepubliceerd via GitHub Actions.

De vernieuwde versie bevat afzonderlijke vacaturedetailpagina's, een sollicitatieformulier met cv-upload, verzending naar het HR-adres en contactformulieren met aparte velden voor voornaam en familienaam.
